## 🔍 连板梯队分析要点

### 一、梯队四角色模型

| 角色 | 定义 | 特征 |
|------|------|------|
| **龙头** | 板块冲锋先锋，最强核心标的 | 连板打开空间，决定板块高度；分游资主导型（纯情绪）和机构主导型（有逻辑支撑） |
| **跟风** | 龙头追随者 | 与龙头同步拉升但高度远不及；完全依附龙头，龙头退潮时率先回调；性价比低 |
| **补涨** | 行情接力者 | 原龙头高位震荡时，资金挖掘新标的；高度受限于原龙头（通常原龙头8板→补涨6板） |
| **中军** | 板块定海神针 | 大市值标的，机构主导；趋势稳步向上；只要中军不垮，板块仍有反复机会 |

### 二、天梯法则（连板高度与行情周期对应）

| 连板高度 | 行情阶段 | 操作策略 |
|----------|----------|----------|
| ≤3板 | 板块轮动/混沌期 | 只做首板试错，不接力 |
| **4板** | **轮动→主线切换关键节点** | 重点关注 |
| 5板 | 主线资金集中入场 | 确认主线方向 |
| 6板 | 全市场热度提升 | 可考虑一进二 |
| **7板** | **主线正式确认** | 围绕核心标的重仓 |
| ≥8板 | 主升加速期 | 主升浪，持仓或低吸 |

### 三、梯队完整性判断
- **完整梯队** = 龙头 + 中军 + 跟风 + 补涨 → 说明板块有持续炒作价值
- **不完整/一日游**：无完整梯队 → 直接放弃，不浪费时间

### 四、撤退信号（板块情绪见顶）
> 当出现：**中军走弱 + 小弟批量炸板 + 补涨熄火 + 只剩龙头孤军奋战** → 核心撤退信号

## 📈 题材炒作逻辑要点

### 一、主线（题材）完整周期——五阶段模型

```
萌芽/试错期 → 发酵/确认期 → 主升/加速期 → 分歧/震荡期 → 退潮/终结期
（混沌）      （主线确立）   （高潮）      （分化）       （崩塌）
```

| 阶段 | 特征 | 信号 | 策略 |
|------|------|------|------|
| **1. 萌芽/试错期** | 上一轮退潮、冰点；新消息刺激；多题材轮动无持续性；连板高度≤3板；炸板率高 | 零星首板、题材一日游、赚钱效应差 | 多看少动，观望为主 |
| **2. 发酵/确认期** | 龙头突破4-5板；板块形成梯队；涨停家数增多、联动性强 | 龙头连板超预期、板块批量涨停、中军稳定 | 拥抱核心，聚焦龙头 |
| **3. 主升/加速期** | 龙头缩量连板/一字板；板块涨停潮30+；低位补涨疯涨；情绪狂热 | 加速、鸡犬升天、融资激增、散户追高 | 最暴利阶段，但随时准备撤退 |
| **4. 分歧/震荡期** | 龙头首次断板/高位放量；板块分化；跟风股补跌；资金高低切换 | 龙头滞涨、后排掉队、高位震荡、成交量萎缩 | 不盲目抄底，关注补涨龙 |
| **5. 退潮/终结期** | 龙头跌停/连续大跌；板块集体杀跌；亏钱效应炸裂；A字杀、核按钮 | 龙头补跌、跌停潮、指数破位 | 直接空仓，等下一轮冰点 |

### 二、题材启动信号识别
1. **板块涨停潮**：单日涨停家数超过5家，且占据涨幅榜前列 → 游资攻击信号，最直接信号
2. **龙头突破4-5板**：打开板块想象空间
3. **梯队完整性**：出现"龙头+中军+跟风"完整结构
4. **指数共振**：板块与大盘指数同步走强
5. **消息持续发酵**：政策/事件驱动具备持续性，而非一次性利好

### 三、题材持续性评估
- **有持续性**：板块排名持续上升（本周20名→下周10名→第三周前5）
- **无持续性**：单日暴涨冲进前十，第二天跌回原形

### 四、情绪传导底层顺序
> 极致冰点时：**个股先于情绪 → 情绪先于板块 → 板块先于大盘**
> → 冰点期逆势走强的个股，是下一轮行情的潜在龙头

## 💡 2-3个可立即应用的知识点

### 知识点1：天梯法则——连板高度择时
**应用场景**：每天复盘时，先看市场最高连板数，判断当前行情阶段
- 最高连板 ≤3板：只做首板试错，不追高位
- 最高连板 4-5板：重点关注，可能切换主线
- 最高连板 ≥7板：确认主线，重仓核心标的
- **实操**：结合今日数据（津药药业7连板），当前处于主线确认期，可围绕医药/AI算力主线核心标的操作

### 知识点2：梯队完整性筛选——只做有完整梯队的板块
**应用场景**：板块筛选时，判断是否值得参与
- 检查该板块是否有：①龙头（连板最高）②中军（大市值趋势股）③跟风小弟（批量涨停）
- 三者缺一不可，否则是一日游行情，不参与
- **实操**：今日有机硅/化工板块有东岳硅材（龙头，20cm）+ 合盛硅业（中军，全产业链）+ 多只跟风 → 梯队完整，值得关注

### 知识点3：买在分歧，卖在一致
**应用场景**：龙头股的操作节奏
- **买在分歧**：龙头主升途中，首次爆量分歧、分时承接有力、不跌破核心支撑位 → 最佳低吸点
- **卖在一致**：龙头缩量加速、全市场一致看好、散户疯狂入场 → 最佳卖点
- **本质**：没有经过分歧考验的龙头，随时可能出现A杀；分歧是检验龙头成色的核心标准
- **实操**：当龙头出现高位炸板/放量大阴线时，不恐慌卖出，等待次日开盘确认；如次日不能反包，再减仓
