## 二、情绪周期四阶段判断体系

### 核心框架

```
冰点期 → 启动期 → 发酵期 → 高潮期 → 退潮期 → 冰点期（循环）
```

### 各阶段特征与指标

| 阶段 | 涨停家数 | 炸板率 | 连板高度 | 市场特征 |
|-----|---------|-------|---------|---------|
| **冰点期** | 极少 | >50% | 无 | 绝望气氛，流动性差 |
| **启动期** | 扩散 | <30% | 1-2板 | 资金试探，赚钱效应开始 |
| **发酵期** | 增加 | <30% | 3-4板 | 梯队形成，题材明确 |
| **高潮期** | 峰值 | <20% | 5板+ | 全市场亢奋，龙头拓展空间 |
| **退潮期** | 减少 | >50% | 高位炸板 | 亏钱效应扩散 |

### 关键指标：炸板率

```
炸板率 = 炸板家数 ÷ 涨停家数

< 30%：资金敢封板，情绪积极（主升/启动期）
30%-50%：资金犹豫，分歧明显
> 50%：追高必埋，情绪极差
```

## 三、连板梯队博弈逻辑

### 梯队分层

| 梯队 | 阶段 | 核心逻辑 | 风险特征 |
|-----|------|---------|---------|
| **首板** | 首次涨停 | 资金开始关注信号 | 随机性高，需次日确认 |
| **1进2** | 第2板 | 题材认可度验证 | 淘汰率高，需题材+资金双驱动 |
| **中位股** | 3-5板 | 资金分歧加剧 | 容易炸板，跟风不稳 |
| **高位股** | 5板+ | 龙头拓展空间 | 纯情绪博弈，需市场支撑 |

### 晋级率规律（参考广发证券策略）
- 各梯队涨停晋级率会中期维持**相对稳定的分布区间**
- 实际晋级率受：题材认可度 + 市场情绪周期 扰动
- 新题材启动时，晋级率可能高于历史均值

## 四、题材梯队识别要点

### 判断维度
1. **龙头股带动**：看跟风股是否跟随
2. **新题材 vs 老题材轮动**：新题材持续性更强
3. **梯队完整性**：是否有1进2、2进3的完整链条
4. **资金集中度**：资金高度集中于龙头 vs 分散于跟风

### 题材炒作三个阶段
1. **预期阶段**：消息刺激，资金提前布局
2. **爆发阶段**：龙头涨停，带动板块
3. **兑现阶段**：高位震荡，资金出逃

## 五、可立即应用的交易知识点

### ✅ 知识点1：炸板率作为情绪温度计

**应用方法**：
- 每日复盘计算炸板率（炸板家数/涨停家数）
- 炸板率<30%时：积极参与连板股
- 炸板率>50%时：停止打板，减少亏损

**实操指标**：
```
开盘30分钟内炸板率 > 40% → 当日不做高位板
尾盘炸板率突然上升 → 次日警惕退潮
```

### ✅ 知识点2：连板梯队择时

**应用方法**：
- **启动期（冰点转暖）**：重仓首板和1进2
- **发酵期（连板扩散）**：参与中位股（3-5板）
- **高潮期（情绪顶峰）**：只做龙头，不追跟风
- **退潮期（高位炸板）**：空仓等待

**择时口诀**：
> "退潮不做板，高潮只做龙，冰点找首板，发酵跟梯队"

### ✅ 知识点3：游资席位联动策略

**应用方法**：
- 同一个游资席位在多只个股买入 → 说明该席位看好某题材
- 章盟主买入+机构买入 → 双重背书，可信度更高
- 净买入额突然放大 → 当日该席位可能持续买入同题材
