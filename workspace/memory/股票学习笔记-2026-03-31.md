## 二、情绪周期六个阶段

```
绝望 → 犹豫 → 乐观 → 疯狂 → 幻灭
  ↓
冰点 → 启动 → 发酵 → 高潮 → 分歧 → 退潮
```

### 各阶段信号特征：

| 阶段 | 涨停家数 | 连板高度 | 亏钱效应 |
|------|----------|----------|----------|
| 冰点 | 极少 | 极低 | 恐慌抛售 |
| 启动 | 增多 | 打开 | 减少 |
| 发酵 | 持续增加 | 拓展 | 消失 |
| 高潮 | 峰值 | 极高 | 出现分歧 |
| 退潮 | 减少 | 压缩 | 扩散 |
| 分歧 | 进一步减少 | 高度下降 | 明显 |

## 三、大盘择时三个安全信号

### ✅ 可以观察的信号：

1. **流动性观察**
   - 央行持续净投放 → 资金面宽松，可适当做多
   - 央行持续净回笼 → 资金面收紧，需谨慎观望

2. **北向资金趋势**
   - 连续3日净流入 → 情绪修复，可考虑加仓
   - 连续3日净流出 → 控制仓位观望

3. **个股涨跌家数**
   - 上涨家数超3000只 → 市场回暖，可积极
   - 上涨家数低于1000只 → 控制风险，减仓

## 五、实战心法（来自游资经验）

### 鹏叔心法：
> **冰点不绝望，启动不犹豫，高潮不贪婪，退潮不硬扛**

### 震荡市五不原则：
1. ❌ 不追单日大涨的热点
2. ❌ 不割在情绪冰点
3. ❌ 不重仓单一板块
4. ❌ 不相信"抄底""翻倍"话术
5. ❌ 不频繁交易

### 逆向思维：
> A股是偏情绪化的，每年都有恐慌性下跌的"撒钱机会"：
> - 2020年2月、2022年4月、2024年2月、2025年4月
> - 恐慌后往往是指数中期行情

## 六、今日总结：2个可行动知识点

### ✅ 知识点1：情绪周期判断
- **应用场景**：每日收盘后判断市场所处阶段
- **操作建议**：
  - 退潮期：极致轻仓或空仓
  - 冰点期：小仓试错，不重仓
  - 启动→发酵期：重仓吃主升
  - 高潮期：分批减仓

### ✅ 知识点2：三个安全信号择时
- **应用场景**：每日盘前/盘后决策
- **操作建议**：
  - 三个信号同时偏多 → 积极做多
  - 三个信号同时偏空 → 减仓观望
  - 信号矛盾 → 保持中立，小仓位试错
