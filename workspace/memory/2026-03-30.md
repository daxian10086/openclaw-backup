### 自动更新功能
- 在app.json中添加version字段
- app.js从app.json读取版本号到globalData
- index.js在onLoad中从globalData读取版本号
- index.wxml使用动态版本号{{version}}
- 页面加载时自动调用checkUpdate()检查更新

### 版本号更新步骤
1. 修改 `app.json` 中的 `version` 字段
2. 修改 `project.config.json` 中的 `versionName` 字段
3. 修改 `app.js` 中 `globalData.version` 的默认值
4. 修改 `pages/index/index.js` 中 `data.version` 的默认值
5. 上传小程序

### 技术问题解决
1. v57导致主界面空白：数据格式不兼容（对象 vs 数组）
2. 版本号不更新：onLoad函数重复定义，导致checkUpdate未执行
3. 版本号显示错误：require('./app.json')可能读取失败，需要在globalData设置正确默认值

### 关键代码结构
```javascript
// app.js
globalData: {
  version: 'v2.3.62',  // 默认值
}

onLaunch() {
  try {
    const appConfig = require('./app.json')
    if (appConfig && appConfig.version) {
      this.globalData.version = appConfig.version
    }
  } catch (e) {
    console.log('读取app.json失败，使用默认版本号', e)
  }
}

// pages/index/index.js
data: {
  version: 'v2.3.62',
}

onLoad() {
  const app = getApp()
  if (app && app.globalData && app.globalData.version) {
    this.setData({ version: app.globalData.version })
  }
  this.checkUpdate()  // 自动检查更新
}

// pages/index/index.wxml
<text class="version">{{version}}</text>
```
