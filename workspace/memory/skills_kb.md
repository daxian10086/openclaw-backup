## 📚 已安装技能列表

### 股票分析 ⭐
- **名称**: stock-analyze
- **用途**: 股票技术分析、短线交易、情绪周期判断
- **使用方法**: 调用 stock-analyze skill 进行股票分析
- **数据源**: 腾讯财经、Baostock
- **常用命令**: 分析指定股票代码

### 图片生成
- **名称**: coze-image-gen
- **用途**: 从文本描述生成图片
- **使用方法**: image_generate 工具
- **参数**: prompt, aspectRatio 等

### 语音合成
- **名称**: coze-voice-gen
- **用途**: 文字转语音（TTS）
- **使用方法**: tts 工具

### 网页抓取
- **名称**: coze-web-fetch
- **用途**: 抓取网页内容（支持 PDF、Office、图片等）
- **使用方法**: coze-coding-dev-sdk web-fetch

### 网络搜索
- **名称**: coze-web-search
- **用途**: 搜索网络信息
- **使用方法**: coze-coding-dev-sdk web-search

### 定时任务
- **名称**: scheduler
- **用途**: 创建和管理定时任务
- **使用方法**: scheduler 工具

### 飞书文档
- **名称**: feishu-doc
- **用途**: 飞书文档读写
- **使用方法**: feishu 相关工具

### 自我改进
- **名称**: self-improving-agent
- **用途**: 捕获学习、错误和改进
- **使用方法**: 在错误、纠正或发现新能力时调用

## 🔧 常用工作流程

### 股票分析流程
1. 使用 stock-analyze skill 获取股票数据
2. 分析技术指标（KDJ、MACD、RSI）
3. 判断情绪周期和位置
4. 生成分析报告

### 文章抓取流程
1. 使用爬虫获取文章列表
2. 使用 coze-web-fetch 获取文章内容
3. 使用 AI 分析提取关键信息
4. 保存到知识库

### 晨报生成流程
1. 获取昨日收盘数据
2. 获取今日开盘前瞻
3. 分析热门板块
4. 汇总重要消息
5. 生成报告并发送

## 📝 技能使用最佳实践

1. **优先使用 API**: 比 UI 操作快 10 倍
2. **错误处理**: 捕获并记录错误
3. **进度跟踪**: 实时写入进度，不要攒到最后
4. **子任务拆分**: 长任务拆分为独立子任务
5. **检查清单**: 建立任务检查清单
