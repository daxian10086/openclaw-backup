## 📖 项目介绍

**mootdx** 是一个用于读取通达信数据的 Python 库。

**通达信**是中国最流行的股票交易软件之一，提供详细的行情数据、K线数据、财务数据等。

---

## 🎯 主要功能

### 1. **数据读取**
- 读取通达信日线数据
- 读取通达信分钟线数据
- 读取财务数据
- 读取板块数据

### 2. **数据格式支持**
- 支持通达信标准数据格式
- 支持日线数据（.day 文件）
- 支持分钟线数据（.lc1、.ln1 等文件）
- 支持扩展数据格式

### 3. **便捷接口**
- 提供 Python 友好的 API
- 简化数据读取流程
- 支持批量处理

## 🔧 与 AKShare 的对比

| 特性 | mootdx | AKShare |
|------|--------|---------|
| **数据来源** | 通达信本地数据 | 网络API（东方财富等） |
| **实时性** | 取决于通达信更新 | 实时数据 |
| **安装** | 需要通达信软件 | 直接安装 |
| **使用门槛** | 较高（需要通达信） | 较低（网络请求） |
| **数据范围** | 本地数据源 | 全市场数据 |
| **适用场景** | 离线分析 | 实时/在线分析 |

## 💡 适用场景

### 适合使用 mootdx：
1. ✅ 已经使用通达信软件
2. ✅ 需要本地离线分析
3. ✅ 有自定义的通达信数据
4. ✅ 需要批量处理本地数据

### 适合使用 AKShare：
1. ✅ 需要实时数据
2. ✅ 不想安装通达信
3. ✅ 需要全市场数据
4. ✅ 网络环境良好

## 🔍 简单示例

```python
from mootdx.quotes import Quotes

# 创建客户端
client = Quotes.factory(market='std')  # 标准市场

# 获取股票数据
data = client.bars(symbol='000001', frequency=9)  # 日线数据
print(data)
```
