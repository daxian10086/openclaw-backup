# 任务检查清单

**创建时间**: 2026-03-25
**用途**: 确保任务执行质量，避免遗漏步骤

---

## 📋 通用任务检查清单

### 任务开始前
- [ ] 明确任务目标
- [ ] 确认所需工具和技能
- [ ] 评估任务复杂度（是否需要拆分子任务）
- [ ] 预估所需时间和资源
- [ ] 向用户确认方案

### 任务执行中
- [ ] 按步骤执行，不要跳过
- [ ] 实时记录进度
- [ ] 遇到问题及时沟通
- [ ] 每完成一步打勾确认
- [ ] 保存中间结果

### 任务完成后
- [ ] 验证结果是否符合预期
- [ ] 总结执行过程
- [ ] 记录遇到的问题和解决方案
- [ ] 更新相关文档
- [ ] 向用户汇报结果

## ✅ 质量控制检查点

1. **准确性**: 结果是否准确？
2. **完整性**: 是否有遗漏？
3. **一致性**: 格式和风格是否一致？
4. **可读性**: 是否易于理解？
5. **可追溯性**: 过程是否可追溯？

---

## 🔄 失败重试机制

- [ ] 等待 60 秒
- [ ] 检查失败原因
- [ ] 尝试修复问题
- [ ] 重试操作
- [ ] 如仍失败，记录详细错误信息

---

**使用说明**: 每次执行任务前，先完成相关检查清单项，执行时实时打勾。
