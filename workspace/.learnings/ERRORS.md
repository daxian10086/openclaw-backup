# Errors

This file captures command failures, exceptions, and error conditions.

## Format
```markdown
## Error: [Title]
**Date:** YYYY-MM-DD  
**Command/Operation:** <what failed>  
**Error:** <error message or output>  
**Context:** <what was being attempted>  
**Root Cause:** <analysis of why it failed>  
**Fix:** <how it was resolved>  
**Prevention:** <how to avoid in future>

**Status:** <resolved | unresolved | recurring>
```
