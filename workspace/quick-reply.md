# 快速回复模板

## 收到消息时的快速回复

当收到用户消息时，使用以下模板第一时间回复：

### ✅ 简单确认（不需要立即行动）
```
收到！
```

### ✅ 正在处理（需要时间）
```
收到，我来检查一下。
```

### ✅ 需要等待
```
收到，稍等片刻。
```

---

## 规则

**每次收到消息后**，在执行任何操作之前：
1. 必须先发送确认回复
2. 然后再开始执行任务
3. 任务完成后再次回复汇报结果

---

## 示例

**用户**：索引做完了没

**我应该回复**：
```
收到，我来检查一下索引状态。
```
然后再执行 `sqlite3` 命令查询。

**用户**：构建失败了

**我应该回复**：
```
收到，我来看看错误信息。
```
然后再执行 `gh run view` 查看日志。
